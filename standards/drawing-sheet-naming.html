Drawing Sheet Naming Convention

Standard file naming structure for all architectural drawing sheets across Flow projects.

Purpose

This naming convention ensures consistent file organization across all Flow projects. It enables quick identification of drawing content, supports efficient searching, and maintains clear revision tracking throughout the project lifecycle.

Structure

(1) FLOW Brand
-
(2) OLY Project
-
(3) DWG Doc Type
-
(4) A Zone
-
(5) AR Discipline
-
(6) 2 Series
-
(7) 01 Descriptor
-
(8) R0 Revision

Components

Each segment serves a specific purpose in the naming hierarchy. Click on a component to learn more about its usage and available options.

1

Flow

FLOW

Brand identifier. Fixed value for all Flow projects to ensure consistent branding across deliverables.

Options

FLOW (Fixed)

Always use "FLOW" to maintain brand consistency across all project documentation.

2

Project Code

OLY

3-letter code identifying the specific project. Assigned by project management at project initiation.

Examples

OLY - Olaya MIA - Miami ATL - Atlanta

Contact the Project Manager for new project code assignments.

3

Document Type

DWG

Identifies the type of document or deliverable being produced.

Options

DWG - Drawing SPEC - Specification CALC - Calculation RPT - Report

Select the type that best matches the document's primary purpose.

4

Zone / Volume

A

Building zone, volume, or system identifier for organizing drawings by location.

Examples

A - Building A B - Building B P - Podium G - Garage

Use single letters for main zones. Sub-zones use A1, A2 format.

5

Discipline

AR

Engineering or design discipline responsible for the drawing content.

Options

AR - Architecture ST - Structural ME - Mechanical EL - Electrical PL - Plumbing FP - Fire Protection ID - Interior Design LA - Landscape ELV - Low Voltage

Match discipline to lead consultant for the drawing content.

6

Sheet Series

2

Drawing series number indicating the type of content shown.

Options

0 - General 1 - Plans 2 - Elevations 3 - Sections 4 - Enlarged Plans 5 - Interior Elev. 6 - RCP 7 - Roof 8 - Schedules 9 - Details

Follow standard AIA sheet organization conventions.

7

Descriptor

01

Sequential sheet number within the series for ordering and reference.

Format

01 - 99

Start at 01 for each series. Leave gaps (01, 05, 10) for future sheets.

8

Revision

R0

Current revision status tracking changes throughout the project.

Options

R0 - Initial Issue R1, R2... - Revisions P1, P2... - Preliminary

Increment revision with each formal issue. Use P for draft submissions.

How to Read a Sheet Name

Each segment provides specific information about the drawing. Here's how to decode a complete file name:

FLOW-OLY-DWG-A-AR-2-01-R0

"This is a Flow project drawing for Olaya, specifically a Drawing for Building A, prepared by Architecture, showing Elevations (series 2), sheet 01, at initial issue."
